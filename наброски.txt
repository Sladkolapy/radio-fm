веб-приложение прослушивания музыки для концентрации, тренировок, звуков для медитации

нужен яркий веб-интерфейс, простота в использовании.
Админка: графики посещаемси, графы для просмотра динамики, путь /admin, уведомления от сообщейий пользователей
архитектура next js, , бд postgres,  бд для музыки яндекс object db, админка chart js, гексагональная архитектура
Описание. Клиент заходит на страничку, регистрируется, придумывая имя и пароль, авторизация сессионная, генерируется токен, который показывается один раз, для того чтобы можно было сбросить пароль. пользователь выбирает категорию и нажимает большую копку проигрования в данной категории музыки, он также может переключаться между категориями и треками. В поиске ему предлагаются наши теги для подбора музыки на его вкус, но поиск только по тегам, далее по этим тегам ищется музыка, которую пользователь может также запустить и она будет идти одна за другой в рандомном порядке. Музыка всегда в рандомном порядке воспроизводится.

В админке можно добавить песню или звук, ее название, добавить ее в определенную категорию, добавить тегов для описания. В админке можно добавть категорию и подкатегорию для музыки, а также сколько угодно тегов. вся музыка находится в Yandex Object Storage, поэтому в .env нужно сразу указать переменные для этого. Проект будет выложен на versel, поэтому бд основная будет на  neon, для нее тоже в .env переменные. плюс нужно пару двигающихся красивых картинок для фона на каждую категорию и подкатегорию, которые будут рандомно отображаться. Создать их и добавить после можно после создания и при создании категорий. Все что я перечислил может удаляться редактироваться и созадваться новое.

В приложение клиент может описать свою просьбу, пожелание, все что угодно и отправить это разработчику, соответсвенно этот разработчик может ответить персонально этому клиенту в оповещениях, также админ может создавать оповещения для определенных пользователей или для  всех, например сообщение с обновлением. У клиента есть маленький колокольчик в правом верхнем углу, который при оповещении будет гореть. В приложении есть возможность перевести пожертвования разработчикам.
